Summary

The purpose of this project was to analyze weather changes as you get closer to the equator. To accomplish this analysis, we first pulled data from OpenWeatherMap API to assemble data for over 500 cities. After assembling the dataset, we used Matplotlib to plot various aspect of the weather vs. latitude. Factors we looked at included: temperature, humidity, cloudiness, and wind speed. This site provides the source data and visualizations as part of the analysis, as well as explanations and descriptions of any trends and correlations witnessed.
